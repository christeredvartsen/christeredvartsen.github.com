Starzinger PEAR channel

Registering the channel: 
pear channel-discover pear.starzinger.net
Listing available packages: 
pear remote-list -c stz
Installing a package: 
pear install stz/package_name
Installing a specific version/stability: 
pear install stz/package_name-1.0.0
pear install stz/package_name-beta
Receiving updates via a feed: 
http://pear.starzinger.net/feed.xml

PHP_BitTorrent - Components for working with torrent files and the BitTorrent format in PHP

SqlDiff - PHP library that creates a series of SQL statements based on two different schemas

SqlDiff is a set of components that can be used to generate a series of SQL statments to make one schema equal to another. Currently only MySQL is supported.
